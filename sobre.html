Tech News

Este projeto faz parte dos exercícios propostos no módulo 4 do Curso de HTML e CSS do Curso em Vídeo.

O objetivo era criar um modelo de site responsivo (que se adapta bem em diferentes tamanhos de tela) e mobile-first (projetado primeiro para dispositivos móveis).

Escolhi criar um site de notícias de tecnologia com visual dos jogos retrô.
